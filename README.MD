### Хай всем кто це читает.
Енто получается главная страница респака с тотемами Империума.

#### Если вкратце, то тупо переименовываете тотем в один из доступных ников.
Их вы можете лицезреть ниже:
* AngryTortik
* M4G1STR
* deesiigneer
* Feyhui
* Old_PC
* sarcasming

Что бы не заморачиваться с постоянным скачиванием респака когда он обновляется, в архив так-же закинут мод для автообновлений. Никаких зависимостей для него не надо, тупо закидываете в папку **mods**. [*У него кстати тоже есть своя страничка на GitHub.*](https://github.com/lexize/ResPackAutoUpdater "GitHub")

Ссылка для скачивания пака: [кликъ](https://github.com/lexize/imperium_totems/raw/main/ImperiumTotems.zip)
